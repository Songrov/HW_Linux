1.Для того, чтобы скопировать на свой компьютер репозитория надо сначала сгенерировать ключ командой ssh-keygen, потом надо привязать ключ Github.
2. Все изменения на GitHub можно получить на локальный компьютер с помощью команды «git pull»
3. git status нам показывает на какой ветке мы находимся  и какие файлы еще не закомитены
Git log  показывает Имя и электронная почта автора коммита,  Уникальный идентификатор, Дата и время коммита.
Git remote -v используется для вывода списка удаленных репозиториев
